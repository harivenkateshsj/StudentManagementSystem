# Student Management System
part 1: 
The Student Management System is a console-based Java application developed using Object-Oriented Programming (OOP) principles. It allows users to register students, enroll them in courses, search for students by their ID, display all registered students, and sort students alphabetically by name.

This project demonstrates the implementation of core Java concepts such as classes, objects, encapsulation, collections, exception handling, regular expressions, lambda expressions, streams, and custom sorting. It is designed as a beginner-friendly project for practicing Java fundamentals and preparing for coding interviews.

Features
Register a new student with validation
Validate student name, age, and student ID
Enroll students in predefined courses
Prevent duplicate course enrollment
Search students using Student ID
Display complete student information
Sort students alphabetically by name
Handle invalid inputs using exception handling
Use Java Streams and Lambda expressions for searching and sorting
Technologies Used
Java
Object-Oriented Programming (OOP)
Collections Framework (ArrayList, List)
Java Streams API
Lambda Expressions
Comparator Interface
Regular Expressions (Regex)
Exception Handling
